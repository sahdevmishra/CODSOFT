# Rule-Based ChatBot

## Description
This is a simple chatbot created using Python. It responds to user input using predefined if-else conditions.

## Features
- Greets the user
- Answers basic questions
- Ends the chat when the user types "bye"

## Technology
- Python

## How to Run
1. Open chatbot.py in Pydroid 3.
2. Run the program.
3. Type your message.

# Tic-Tac-Toe AI

## Description
This project is a simple Tic-Tac-Toe game developed using Python. The user plays against the computer, which makes decisions automatically based on simple AI logic.

## Features
- Human vs Computer gameplay
- Checks for win, loss, and draw
- Simple AI that blocks and plays winning moves

## Technology Used
- Python

## How to Run
1. Open tic_tac_toe.py in Pydroid 3.
2. Run the program.
3. Enter positions from 1 to 9 to play.
